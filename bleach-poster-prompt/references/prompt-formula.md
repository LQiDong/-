# Prompt Formula

Use this formula when generating a BLEACH / 死神 poster prompt.

## Ordering rule

Keep the prompt in this order:

1. Format and composition
2. Main subject silhouette
3. Group characters inside the silhouette
4. World-setting and signature locations
5. Symbols and props
6. Color palette, texture, and atmosphere
7. Typography system
8. Negative constraints

## Default composition recipe

- Vertical poster
- Large side-profile silhouette as the outer contour
- Inner world rendered as silhouette-fill narrative composition
- Strong group-cast presence, not a solo portrait
- Large negative space and restrained layout

## Default BLEACH anchors

Use these when the user does not specify alternatives:

- Main silhouette: 黑崎一护
- Core supporting cast: 朽木露琪亚、石田雨龙、茶渡泰虎、井上织姬、阿散井恋次、朽木白哉、蓝染惣右介
- World anchors: 尸魂界、静灵庭、虚圈、黑腔、穿界门、斩魄刀、灵压、白色面具碎片、月光、樱花、战场遗迹

## Style anchors

- cinematic poster
- dreamlike watercolor illustration
- paper grain
- brush marks and feathered edges
- soft atmospheric perspective
- light mist transitions
- restrained, sacred, poetic, legendary mood
- rich but coherent palette
- default to vivid accents such as sakura pink, cobalt blue, teal glow, violet haze, warm gold, and small crimson highlights when the theme supports them
- avoid flattening every poster into a mostly black-and-blue look unless the user explicitly asks for that

## Text system options

Pick one:

- no large title, only poster-like side copy and corner metadata
- bottom title text with a heavy cinematic Chinese type treatment
- vertical slogan at the edge plus a small information block in a corner

## Safe negative prompt

Use a negative prompt like:

- no generic fantasy background
- no random collage
- no clutter
- no cheap stock-art feel
- no modern neon overload
- no flat poster template
- no watermark
- no logo

## Reusable prompt skeleton

```text
为《死神 / BLEACH》创作一张竖版收藏级海报，采用[构图]。
主体是[主角]的巨大侧脸剪影作为外轮廓；剪影内部自然生长出[群像角色]与[世界场景]，
并包含[象征符号和道具]，形成高级的轮廓填充式叙事合成。
整体气质为[气质关键词]，风格融合电影海报与梦幻水彩插画，
使用[丰富但协调的色彩]、纸张颗粒、飞白、刷痕、轻雾化过渡与大面积留白。
[文字系统说明]。
负面约束：不要[负面项]。
```

## Example prompt block

```text
为《死神 / BLEACH》创作一张竖版收藏级海报，采用左侧巨大侧脸剪影 + 中央群像叙事填充 + 边缘留白的构图。
主体是黑崎一护的侧脸剪影；剪影内部自然生长出朽木露琪亚、石田雨龙、茶渡泰虎、井上织姬、阿散井恋次、朽木白哉、蓝染惣右介等主要角色，
并包含尸魂界、静灵庭、虚圈、黑腔、穿界门、斩魄刀、灵压、白色面具碎片、月光、樱花、战场遗迹，
形成高级的轮廓填充式叙事合成。整体气质安静、宏大、神圣、怀旧、诗意、传说感，
风格融合电影海报与梦幻水彩插画，使用冷白、靛蓝、灰银作为基底，
并加入樱花粉、暖金、青绿灵光、少量血红与光晕点缀，搭配纸张颗粒、飞白、刷痕、轻雾化过渡与大面积留白。
文字系统采用海报式竖排标语与角落信息块，不要大标题，不要水印，不要logo，不要杂乱拼贴，不要廉价奇幻素材感。
```
