## 01. AI 辅助投资决策系统

- 项目定位：面向个人投资研究场景的 AI 辅助分析工具。
- 核心问题：把分散的行情、情绪、规则和风险信号整合成可评估的决策链路。
- 产品方法：
  - 规则引擎识别基础交易条件。
  - LLM 做新闻与市场情绪分析。
  - 数据置信度分级，避免把低可信信息直接包装成结论。
  - 通过历史交易日样本做评测与 badcase 归因。
- 页面关键词：规则引擎、LLM 情绪分析、数据置信度、badcase 迭代。
- 风险边界：该项目展示 AI 产品设计与评测方法，不作为真实投资建议。

## 02. AI 广告素材诊断工具

- 项目定位：面向投放和内容运营场景的 AI 素材诊断工具。
- 核心问题：帮助运营人员快速判断广告素材的表达问题、平台规则风险和优化方向。
- 产品方法：
  - Prompt Chain 拆解素材诊断流程。
  - LLM-as-a-Judge 生成结构化评估。
  - 三层规则库覆盖平台规范、内容质量、转化表达。
  - 结合跑量表现和平台规则差异做复盘。
- 页面关键词：Prompt Chain、素材诊断、LLM-as-a-Judge、规则库。

## 03. AI 面试训练 Agent

- 项目定位：面向 AI 产品经理求职者的模拟面试训练 Agent。
- 核心问题：让求职者在真实 JD 语境下反复练习表达、结构化回答和复盘。
- 产品方法：
  - JD 解析 Agent 提取岗位要求。
  - 面试官 Agent 生成追问。
  - 评估官 Agent 做能力评分。
  - 教练 Agent 输出优化建议。
  - Judge 复核回答质量，降低单次生成偏差。
- 页面关键词：Agent 工作流、JD 解析、MECE 评分、面试复盘。

## 04. 下一段 AI 产品现场

- 当前用途：作为未来项目入口和求职联系转化点。
- 后续可替换为：
  - 真实实习项目。
  - AI 产品原型项目。
  - 个人作品集中的深度案例页。

# 网站代码与文件说明

## 当前技术栈

- React
- Vite
- Three.js
- 原生 CSS
- Canvas 2D 粒子动画
- Web Audio API

## 运行命令

```bash
npm.cmd install
npm.cmd run dev
npm.cmd run build
```

本地预览地址：

```text
http://127.0.0.1:5173/
```

## 核心交互说明

### Hero 粒子标题

- 主标题由 Canvas 粒子组成。
- 鼠标经过标题时，粒子轻微散开。
- 鼠标移开后，粒子自动聚拢回文字形状。
- 鼠标移动时触发 Web Audio 生成的轻柔八音盒质感旋律。
- 音频为代码合成，不引用外部音乐文件。

### 项目地图

- 地图背景使用插画图：`assets/season-map-illustration.png`
- Three.js 透明叠加在插画背景上。
- 地图有春夏秋冬四个地点。
- 鼠标悬停地点时：小光人沿路线移动到对应地点；地点光圈点亮；白色项目卡片显示。
- 点击地点时：春夏秋跳转到对应项目卡片；冬跳转到底部联系区。

## 后续维护建议

- 如果项目文案要更新，优先修改 `src/App.jsx` 里的 `projects` 数据。
- 如果要替换项目图片，优先把新图片放进 `assets/`，再在 `src/App.jsx` 中更新引用路径。
- 如果要调整地图视觉，优先替换 `assets/season-map-illustration.png`，再微调 `src/styles.css` 和 `ProjectMap3D` 里的地点坐标。
- 如果要增加新项目，需要同步更新 `projects`、`seasonSpots` 和项目展示区。
